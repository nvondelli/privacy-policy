Política de Privacidad

Control Álbum Mundial 2026

Última actualización: Mayo 2026

1. Información que recopilamos

La aplicación Control Álbum Mundial 2026 no recopila ni almacena ningún dato personal del usuario en servidores externos. Toda la información se guarda únicamente en el dispositivo del usuario.

2. Uso de la cámara

La aplicación solicita permiso de cámara únicamente para escanear códigos de láminas. Las imágenes tomadas no se almacenan ni se comparten con terceros.

3. Almacenamiento

La aplicación solicita permiso de almacenamiento únicamente para exportar e importar el archivo de respaldo del álbum. Este archivo permanece en el dispositivo del usuario.

4. Compartir información

La aplicación permite compartir listas de láminas faltantes y repetidas a través de aplicaciones de mensajería del dispositivo. Esta acción es iniciada voluntariamente por el usuario.

5. Servicios de terceros

La aplicación utiliza Google ML Kit para el reconocimiento de texto en imágenes. Este procesamiento se realiza localmente en el dispositivo.
